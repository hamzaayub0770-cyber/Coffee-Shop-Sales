# ☕ Coffee Shop Sales Dashboard

An interactive **Microsoft Excel Dashboard** built to analyze coffee shop sales using Pivot Tables, Pivot Charts, KPI Cards, and Slicers.

---

## 📊 Dashboard Preview

![Dashboard Preview](Dashboard_Preview.png)

---

## 🎯 Project Objective

This project analyzes coffee shop sales data and presents key business insights through an interactive Excel dashboard.

---

## 📈 Dashboard KPIs

- 💰 Total Revenue
- 📦 Total Orders
- 💵 Average Sale
- 🏆 Highest Sale
- ☕ Best Selling Product
- 📍 Sales by Store Location
- 📅 Orders by Weekday

---

- Data Analysis
